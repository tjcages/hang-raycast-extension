# Hang Changelog

## [1.0.0] - 2024-12-19

### Added
- Initial release of Hang extension
- Support for creating Google Meet meeting links
- Support for creating Zoom meeting links
- Auto-copy meeting links to clipboard
- Option to auto-open links in browser
- Platform selection in preferences (Google Meet or Zoom)
- OAuth integration for Google Calendar API
- Server-to-Server OAuth support for Zoom API
